The main objective of the project is:

  • Development of a framework which is flexible and fast.
  • Implementation of an analytical pipeline in a HPC ecosystem which will be rendered easily accessible and usable to the national and international research community through a webserver.

We shall develop flexible computational framework in high performance compute environment for rapidly processing sequence data in a highly parallelized way. This HPC based analysis framework would be available to the research community via secure remote access.

  • Aim of the project :

    To create a framework for assembling scripts into an automated, version controlled, parallelized pipeline pipeline.

  • The benifits of the project :

    • Reduction of human resources required for data management by introducing automated processes.
    • Reduction of time by executing all operations more reliably and quickly.
    • Storing all information in one repository with access control and search capabilitie.
    • A non-programmer can easily execute the supported pipelines.

opportunities

We will deliver a highly flexible HPC aware analysis framework for rapid processing of massively parallel high throughput sequence data. The analysis framework would also allow to Indian researchers to directly login to the framework and access NGS analysis pipelines on demand.

Click to HPC

The analysis platform would provide opportunities to biologist and early career bioinformaticians with little experience in UNIX command line to run complex pipeline in the high performance server scale architecture in the select-click-go-analyze approach.The platform would allow the advance users to customize the options and parameters of the chosen tools.

Easy NGS

The analysis platform would allow basic operations like read QC, filtering, alignment, variant calling (SNP, indel, CNV and SVs), quantifying expression and downstream analysis like variant annotation and testing for differential expression from a front-end web portal supported by a highly parallelized background processing core.

Cutting edge analysis

The analysis platform would also allow user to do three broader type of downstream analysis on VCF files - (i) pedigree based analysis, (ii) analysis of somatic mutations, (iii) population genetics analysis - disease association, ancestry, evolutionary relationship and natural selection.

Customizable workflow

Advance users of the platform would have the choice to edit workflow configuration files and use customized parameters specific to individual tool. User would also get an opportunity to build new workflow from tools in the tools repository.
